# Antigravity Skill System - 서비스 오픈 로드맵

현재 프론트엔드 UI/UX와 **로컬 IDE 자동 설치(File System Access API)** 기능이 매우 성공적으로 구현되었습니다. 실제 사용자들에게 서비스하기 위한 **"정식 오픈(Service Launch)"**까지 남은 작업들과, **당장 오늘부터 진행해야 할 작업**을 정리했습니다.

---

## 🚀 시스템 서비스 오픈까지 남은 전체 로드맵 (Roadmap)

정식 서비스 오픈을 위해서는 아래 4가지 핵심 인프라가 완성되어야 합니다.

### Phase 1: 리얼 데이터 파이프라인 (백엔드 연동)
현재는 화면에 보여주기 위한 '가짜(Mock) 스킬 데이터'를 사용 중입니다. 
- GitHub 등 외부 저장소에서 실제 스킬들을 긁어오는(Crawling) 기능 구현
- 스킬 원본(영어)을 번역 API를 통해 **한글로 자동 번역**하여 DB(또는 JSON)에 구축
- 사용자가 직접 자신의 스킬을 등록할 수 있는 폼 구현

### Phase 2: 진짜 AI 스마트 추천 엔진 연동
현재 'AI 추천받기' 버튼은 단순 키워드 매칭으로 임시 동작하고 있습니다.
- 실제 LLM(Gemini / Claude API)을 연동하여, 사용자의 자연어 프롬프트를 분석하고 가장 적합한 스킬을 추론해 내는 로직 개발

### Phase 3: GitHub URL -> 스킬 변환기 (Core 로직)
사용자가 임의의 GitHub Repo URL이나 Claude 시스템 프롬프트를 입력하면, 이를 파싱하고 Antigravity/Cursor/Windsurf 호환 포맷으로 즉시 변환해 주는 메인 로직 고도화.

### Phase 4: 호스팅 및 배포 (Service Open)
- Vercel, Netlify 또는 Google Cloud를 이용해 정적 웹사이트 호스팅
- 실제 도메인 연결 및 SEO(검색엔진 최적화) 세팅
- 서비스 정식 런칭

## 🔥 현재 당장 개발해야 할 부분 (What to do NOW)

UI 작업이 마무리되었으므로, 이제 **"실제 데이터와 기능"**을 붙이는 작업을 시작해야 합니다. 다음 두 가지 중 하나를 먼저 진행하는 것을 추천드립니다.

### 옵션 A: 진짜 AI 엔진 붙이기 (추천)
현재 화면에 만들어둔 **[✨ AI 스마트 스킬 추천]** 검색창이 실제로 동작하도록, LLM API(예: Gemini API)를 연결하여 사용자가 질문하면 진짜 AI가 스킬을 골라주는 백엔드 로직을 구현합니다. 서비스의 가장 핵심적이고 재밌는 기능입니다.

### 옵션 B: 스킬 마켓 리얼 데이터 연동
하드코딩된 스킬 목록을 지우고, 외부 JSON 파일이나 데이터베이스에서 스킬 목록을 동적으로 불러오도록 구조를 변경합니다. 이후 스킬 크롤링 스크립트를 작성하여 스킬 개수를 대폭 늘립니다.
